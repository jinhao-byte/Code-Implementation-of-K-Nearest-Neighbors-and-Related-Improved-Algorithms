# Code-Implementation-of-K-Nearest-Neighbors-and-Related-Improved-Algorithms
K近邻算法是机器学习与数据挖掘领域的经典算法，使用Java复现了该经典算法与其相关改进算法。
上述代码可以在WEKA平台上运行。
WEKA仓库：https://github.com/Waikato/weka-3.8

K近邻算法：
Guo, Gongde, et al. "KNN model-based approach in classification." OTM Confederated International Conferences" On the Move to Meaningful Internet Systems". Berlin, Heidelberg: Springer Berlin Heidelberg, 2003.
相关改进算法：
EkNN：
Denoeux T. A k-nearest neighbor classification rule based on 
Dempster-Shafer theory[J]. IEEE Transactions on Systems, Man, 
and Cybernetics, 1995, 25(5): 804-813.
PNN：
Zeng Y, Yang Y, Zhao L. Pseudo nearest neighbor rule for 
pattern classification[J]. Expert Systems with Applications, 2009, 
36(2): 3587-3595.
MkNN：
Parvin H, Alizadeh H, Minaei-Bidgoli B. MKNN: Modified 
k-nearest neighbor[C]. In: Proceedings of the World Congress on 
Engineering and Computer Science (WCECS). 2008, pp: 1-4.
